# NextItems (Next.js + Express)

A simple Next.js (App Router) application with:
- Public landing page (7 sections)
- Public item list + item details pages
- Mock cookie-based login
- Optional protected "Add Item" page that stores items via an Express API (JSON file)

## Tech
- Next.js 15/16 (App Router)
- Express.js
- TailwindCSS
- Cookies for mock auth
- react-hot-toast (toast notification)
